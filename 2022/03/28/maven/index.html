Conception

1.1 仓库

存储资源，各种jar包

  • 本地仓库：/Users/yus/.m2/repository
  • 远程仓库:
    • 中央仓库，maven团队维护
    • 私服：公司存储的，具有版权

1.2 坐标

描述仓库中资源的位置

https://repo1.maven.org/maven2

  • groupId: maven项目隶属组织名称（org.mybatis)
  • artifactId: maven项目的名称
  • version： 版本号
  • packaging： 打包方式

这个地址指定了中央仓库的地址

/usr/local/apache-maven-3.8.4/lib/maven-model-builder-3.8.4/org/apache/maven/model/pom-4.0.0.xml

因为这个地址的国外的地址，所以我们可以在用户设置里更改central的访问路径，通过镜像来实现

mvn compile 生成targe文件夹，class文件

mvn clean 删除target文件夹

mvn package 打包 ：compile ⇨ testCompile test ⇨ package
