workbench在win10下无法启动问题

Posted on 2021-01-05 Edited on 2021-03-28 In tech

好久没编程果然出现了很多问题，下面是一些记录。

workbench无法启动

  1. 首先用命令行启动MySQL进行检查是否是MySQL的问题

    • mysql -u 用户名 -p
    • 结果登录失败

  2. 查看是否开启MySQL服务

    • 打开“服务”启动MySQL服务
    • 发现是这个问题，重新登录成功，说明不是MySQL的问题

  3. 右键开始菜单，打开“事件查看器”，查看日志

  4. 发现错误.NET Runtime error,想到也许是workbench的Requeryments缺失，遂去官网搜索缺什么（https://dev.mysql.com/doc/workbench/en/wb-requirements.html）

    1. Microsoft .NET Framework 4.5.2（这个下载之后，无法安装，说已经有更高版本）
    2. Microsoft Visual C++ Redistributable for Visual Studio 2019（安装完这个就可以打开了）
    3. Microsoft Windows 10 or Windows Server 2019

总结

虽然就这几步，但是花了很长时间，总之整理电脑文件时要谨慎对待！
