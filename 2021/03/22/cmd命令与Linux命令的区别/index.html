cmd命令与Linux命令的区别

Posted on 2021-03-22 Edited on 2021-03-28

太长不看总结

cmd是.exe程序,windows系统拿来模拟Dos系统的

Linux命令也就是shell,是用户态于内核态的交互层,功能更强大.

功能 cmd linux
复制文件 copy cp
移动文件 move mv
列举文件 dir ls
清除屏幕 cls clear
退出 exit exit
显示修改日期 date date
删除文件 del rm
回响信息到屏幕 echo echo
用编辑器编辑文件 edit vi
比较文件内容 fc diff
在文件中查找字符串 find grep
显示命令帮助 命令/? man 命令
创建目录 mkdir mkdir
删除目录 rd rmdir
目录更名 move mv
拷贝目录 xcopy copy
查看文件 more less
重命名文件 ren mv
切换目录 cd cd
显示时间 time date
显示已被使用的内存 mem free
在屏幕打印文件内容 type cat
# linux # os
理解BIO,NIO,AIO,IO多路复用
理解强引用、软引用、弱引用、虚引用
